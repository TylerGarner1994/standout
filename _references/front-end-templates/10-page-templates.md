# 10 PAGE TEMPLATES

This document contains 10 page templates for common web page types.
Each template provides a complete section structure with copy direction for each section.

---

## HOW TO USE THESE TEMPLATES

Each template provides:
- **Page Type** - What kind of page this is
- **Best For** - When to use this template
- **Section Structure** - Every section in order with copy direction
- **Conversion Goal** - What action this page drives
- **Key Principles** - What makes this template convert

All templates follow:
- Single-column mobile-first layout
- Clear conversion goal with single CTA
- Social proof throughout (not just at the bottom)
- No em dashes in copy direction examples

## TEMPLATE 1: HIGH-TICKET SERVICE LANDING PAGE

**Best For:** Agencies, consultants, coaching programs ($2K+)
**Conversion Goal:** Book a discovery call

### Section Structure

**SECTION 1: HERO**
```
Headline: [Transformation promise for specific audience]
Subheadline: [Credibility + how it works + timeframe]
CTA Button: "Book Your Free Strategy Call"
Trust Bar: [3 logos or stats below CTA]
```

**SECTION 2: PROBLEM AGITATION**
```
Headline: "If you're [target audience], you know this feeling:"
Content: 5-7 bullet points describing specific pains
(Each bullet = one specific frustration they feel regularly)
Transition: "You've tried [X], [Y], [Z]. Here's why they didn't work."
```

**SECTION 3: ROOT CAUSE**
```
Headline: "The real reason [problem persists]"
Content: 2-3 paragraphs explaining the systemic issue
(Not their fault. The industry/approach is broken.)
Bridge: "That's why we built [solution]."
```

**SECTION 4: SOLUTION INTRODUCTION**
```
Headline: "Introducing [Solution Name]"
Subheadline: [One sentence description]
Content: How it works in 3 steps
Step 1: [Action] → [Result]
Step 2: [Action] → [Result]
Step 3: [Action] → [Result]
```

**SECTION 5: PROOF BLOCK 1**
```
Headline: "Real results from real clients"
Content: 3 case studies in card format
Each card: Name, Business Type, Before State, After State, Timeframe
[PLACEHOLDER: Insert real client results]
```

**SECTION 6: WHAT'S INCLUDED**
```
Headline: "Everything you get:"
Content: Value stack with descriptions
- [Deliverable 1] with brief description
- [Deliverable 2] with brief description
- [Deliverable 3] with brief description
[PLACEHOLDER: Add actual deliverables]
CTA: "Start Today"
```

**SECTION 7: PROOF BLOCK 2**
```
Content: 2-3 longer testimonials with photos
[PLACEHOLDER: Real testimonials with permission]
```

**SECTION 8: OBJECTION HANDLING**
```
Headline: "Common questions:"
Format: FAQ accordion
5-7 questions that overcome real objections
[PLACEHOLDER: Use actual objections from sales calls]
```

**SECTION 9: GUARANTEE**
```
Headline: "Our guarantee:"
Content: Specific guarantee statement
[PLACEHOLDER: Insert actual guarantee terms]
Note: Must be specific. "Satisfaction guarantee" converts poorly.
```

**SECTION 10: FINAL CTA**
```
Headline: "Ready to [transformation]?"
Content: 2-3 sentences summarizing the offer
CTA Button: "Book Your Free Call"
Urgency: [Real capacity constraint if applicable]
```

**Key Principles:**
- CTA appears minimum 3 times (hero, mid-page, bottom)
- Social proof appears in sections 5, 7, and implied throughout
- Objections addressed before final CTA
- Guarantee reduces decision friction at close

## TEMPLATE 2: LEAD MAGNET / OPT-IN PAGE

**Best For:** Free resource, checklist, guide, webinar registration
**Conversion Goal:** Email address capture

### Section Structure

**SECTION 1: HERO**
```
Headline: "Free: [Specific valuable thing] for [Specific audience]"
Subheadline: [What they'll be able to do after getting this]
Visual: Image of the resource (even if digital, make it look tangible)
Form: Name + Email + Button: "Send Me the [Resource]"
Microcopy: "We respect your privacy. Unsubscribe anytime."
```

**SECTION 2: WHAT'S INSIDE**
```
Headline: "Here's exactly what you'll get:"
Content: 5-7 bullets of specific contents
Each bullet: [Specific thing] so you can [specific benefit]
```

**SECTION 3: WHO THIS IS FOR**
```
Headline: "This is for you if:"
Content: 5 bullet points describing the right person
Each bullet: You are/do/want [specific thing]
Closing: "If that's you, this is exactly what you need."
```

**SECTION 4: ABOUT THE CREATOR**
```
Headline: "Why trust this?"
Content: Brief credentials + why this specific knowledge
Keep brief. 100-150 words max.
[PLACEHOLDER: Real credentials only]
```

**SECTION 5: FINAL FORM**
```
Headline: "Get instant access:"
Form: Email + Button
Reassurance: "Joins [X] other [audience type] who've downloaded this"
[PLACEHOLDER: Real download number]
```

**Key Principles:**
- One conversion goal, zero distractions
- No navigation menu (increases conversion by 20-30%)
- Make the free resource feel valuable enough to be sold
- "What's inside" section does the heavy lifting

## TEMPLATE 3: PRODUCT/SAAS HOMEPAGE

**Best For:** Software products, SaaS, digital tools
**Conversion Goal:** Free trial or demo booking

### Section Structure

**SECTION 1: HERO**
```
Headline: [What it does + who it's for + key differentiator]
Subheadline: [How it works + key benefit]
CTA Primary: "Start Free Trial"
CTA Secondary: "Watch Demo" (lighter styling)
Social Proof: "Trusted by X businesses" + logos
```

**SECTION 2: PROBLEM STATEMENT**
```
Headline: "[Current tools/approach] weren't built for [specific use case]"
Content: 3 columns showing current pain
Each column: Icon + pain point title + 1-2 sentence description
```

**SECTION 3: PRODUCT TOUR**
```
Headline: "[Product] was built differently"
Content: Alternating feature sections (left text, right screenshot)
3-4 key features maximum
Each section: Feature name + 2-3 sentence benefit description + screenshot
```

**SECTION 4: SOCIAL PROOF**
```
Headline: "What teams are saying"
Content: 6 short testimonials in 2x3 grid
Each: Quote + Name + Title + Company
[PLACEHOLDER: Real testimonials]
```

**SECTION 5: PRICING**
```
Headline: "Simple, transparent pricing"
Content: 2-3 tier pricing table
Most popular tier highlighted
All tiers include: Price + Feature list + CTA
[PLACEHOLDER: Actual pricing]
```

**SECTION 6: INTEGRATIONS**
```
Headline: "Works with your existing tools"
Content: Logo grid of key integrations
[PLACEHOLDER: Actual integrations]
```

**SECTION 7: FINAL CTA**
```
Headline: "Start [achieving outcome] today"
CTA: "Start Free Trial. No credit card required."
Reassurance: Free trial terms
```

**Key Principles:**
- Screenshot-forward (show, don't just tell)
- Pricing visible on homepage (reduces support load + increases trust)
- Integration logos = lower switching cost anxiety
- "No credit card required" increases trial starts by 30-40%

## TEMPLATE 4: WEBINAR / EVENT REGISTRATION

**Best For:** Live or automated webinars, workshops, masterclasses
**Conversion Goal:** Registration (email + date commitment)

### Section Structure

**SECTION 1: HERO**
```
Event label: "Free Live Masterclass" or "Free Workshop"
Headline: "How to [specific transformation] in [timeframe] [even if objection]"
Date/Time: [Specific date + time + timezone]
Host: Photo + Name + brief credibility
CTA: "Reserve My Spot"
Scarcity (if real): "[X] spots available"
```

**SECTION 2: WHAT YOU'LL LEARN**
```
Headline: "In this [time] [event type], you'll discover:"
Content: 5-7 bullet points of specific learnings
Each bullet: How to [specific thing] that [specific outcome]
```

**SECTION 3: WHO THIS IS FOR**
```
Headline: "This is for you if:"
Content: 5 bullet points describing right attendee
"This is NOT for you if:": 2-3 disqualifiers
(Disqualifiers increase registration quality and show confidence)
```

**SECTION 4: ABOUT THE HOST**
```
Headline: "Meet your host"
Content: Photo + credibility story in 150-200 words
Focus on: Why they know this, what results they've helped others achieve
[PLACEHOLDER: Real bio with verifiable credentials]
```

**SECTION 5: PROOF**
```
Headline: "Here's what past attendees experienced:"
Content: 4-6 testimonials from previous events
[PLACEHOLDER: Real testimonials from actual attendees]
```

**SECTION 6: FINAL REGISTRATION**
```
Headline: "Save your spot:"
Form: Name + Email + "Reserve My Spot" button
Date reminder: [Date + Time]
Reassurance: "Can't attend live? Register anyway and we'll send the replay."
```

**Key Principles:**
- Specific date/time creates real urgency
- Disqualifiers increase quality of registrants
- Replay offer reduces fence-sitter excuse
- Host credibility section is non-negotiable for conversion

## TEMPLATE 5: ECOMMERCE PRODUCT PAGE

**Best For:** Physical or digital products sold direct to consumer
**Conversion Goal:** Add to cart / Purchase

### Section Structure

**SECTION 1: HERO (Above fold)**
```
Left column: Product image gallery (multiple angles)
Right column:
  - Product name
  - Star rating + review count
  - Price (with original price if discounted)
  - Key benefit bullets (3-5 max)
  - Variant selection (size, color, etc.)
  - Add to Cart button (dominant)
  - Trust badges: [Shipping, Returns, Guarantee]
```

**SECTION 2: PRODUCT STORY**
```
Headline: "Why we made this"
Content: 2-3 paragraphs on the problem it solves + how it was designed
Tone: Founder-voice, genuine
[PLACEHOLDER: Real product development story]
```

**SECTION 3: FEATURES AND BENEFITS**
```
Headline: "What makes [product] different"
Content: 3-4 feature blocks
Each block: Icon + Feature name + Benefit description
Focus on: Why it matters to the customer, not specs
```

**SECTION 4: SOCIAL PROOF**
```
Rating summary: Stars + number of reviews + distribution
Featured reviews: 4-6 reviews with photos if possible
Filter options: By star rating, verified purchase
[PLACEHOLDER: Real customer reviews]
```

**SECTION 5: FAQ**
```
Headline: "Questions about [product]:"
Content: 5-7 most common pre-purchase questions
[PLACEHOLDER: Real FAQ based on customer service tickets]
```

**SECTION 6: FINAL CTA**
```
Recap: Key benefit + price + guarantee
Add to Cart button
Shipping and returns reminder
```

**Key Principles:**
- Images do 60% of selling on product pages
- Social proof (reviews) is the most important trust factor
- Trust badges at point of purchase decision
- Shipping/returns must be prominently stated

## TEMPLATE 6: CASE STUDY PAGE

**Best For:** B2B, high-ticket services, complex sales
**Conversion Goal:** Contact / Discovery call booking

### Section Structure

**SECTION 1: HEADLINE**
```
Format: "How [Client] [achieved specific result] in [timeframe]"
Subheadline: [Company type + starting situation + key metric improvement]
CTA: "See If We Can Do This For You"
```

**SECTION 2: THE CHALLENGE**
```
Headline: "The Challenge"
Content: 200-300 words describing:
- Who the client was
- What problem they faced
- Why it was a problem
- What they'd tried before
[PLACEHOLDER: Real client challenge with permission]
```

**SECTION 3: THE APPROACH**
```
Headline: "Our Approach"
Content: 3-4 phases of work with description
Focus on: What was done and why, not just what
Include: Timeline markers where possible
[PLACEHOLDER: Real work performed]
```

**SECTION 4: THE RESULTS**
```
Headline: "The Results"
Content: Key metrics in large typography
3-5 metrics: Before + After + Percentage change
Supporting narrative: 150-200 words explaining the results
[PLACEHOLDER: Real verified results only]
```

**SECTION 5: CLIENT QUOTE**
```
Large pull quote from client about results or experience
Attribution: Name, Title, Company
[PLACEHOLDER: Real quote with permission]
```

**SECTION 6: IMPLICATIONS**
```
Headline: "What this means for businesses like [client type]"
Content: 2-3 paragraphs connecting this result to the reader's situation
Bridge: "If your situation is similar..."
```

**SECTION 7: CTA**
```
Headline: "Want similar results for your business?"
Content: Brief description of how to start
CTA: "Book a Free Strategy Call"
```

**Key Principles:**
- Specific numbers throughout (not vague claims)
- Client voice (quote) is the most credible element
- "Implications" section connects story to reader
- Only use real, verifiable results

## TEMPLATE 7: ABOUT PAGE

**Best For:** Any business where trust is important to conversion
**Conversion Goal:** Increase trust + drive to primary CTA

### Section Structure

**SECTION 1: MISSION STATEMENT**
```
Headline: "We exist to [mission in plain language]"
Subheadline: [Who you serve + what you help them achieve]
No jargon. First principles.
```

**SECTION 2: ORIGIN STORY**
```
Headline: "Why we started"
Content: 300-400 words telling the founding story
Must include: The problem the founder experienced, the moment of decision, why this approach, what's different now
Tone: Personal, honest, specific
[PLACEHOLDER: Real founding story]
```

**SECTION 3: THE TEAM**
```
Headline: "The people behind [Company]"
Content: Team member cards
Each card: Photo + Name + Role + 2-3 sentence bio
[PLACEHOLDER: Real team with real photos]
```

**SECTION 4: THE NUMBERS**
```
Headline: "By the numbers"
Content: 4-6 impressive real metrics
Examples: Clients served, revenue generated for clients, years in operation
[PLACEHOLDER: Real verified numbers only]
```

**SECTION 5: VALUES**
```
Headline: "What we believe"
Content: 3-5 genuine values with explanation
Each value: Name + 2-3 sentences explaining what it means in practice
Avoid generic values (integrity, excellence). Be specific.
```

**SECTION 6: SOCIAL PROOF**
```
Headline: "What clients say about working with us"
Content: 4-6 testimonials focused on the experience, not just results
[PLACEHOLDER: Real testimonials]
```

**SECTION 7: CTA**
```
Headline: "Ready to work with us?"
CTA: Primary action button
Secondary: Link to case studies or portfolio
```

**Key Principles:**
- Real photos of real people (stock photos destroy trust)
- Founding story creates emotional connection
- Numbers section adds credibility
- Values section attracts right clients, repels wrong ones

## TEMPLATE 9: 404 / ERROR PAGE

**Best For:** All websites (recovery page)
**Conversion Goal:** Keep user on site + guide to valuable content

### Section Structure

**SECTION 1: ERROR ACKNOWLEDGMENT**
```
Headline: "Hmm, that page doesn't exist."
Subheadline: "But you're in the right place. Here's where to go:"
Tone: Friendly, human, not robotic
```

**SECTION 2: HELPFUL NAVIGATION**
```
Content: 3-5 links to most valuable pages
Format: Card or list with description
Options:
  - Homepage
  - Most popular blog post or resource
  - Main product/service page
  - Contact page
```

**SECTION 3: SEARCH**
```
Content: Search bar if site has search
Placeholder: "What were you looking for?"
```

**SECTION 4: BRAND MOMENT**
```
Optional: Use this space for a brand-appropriate message or visual
Make the error page feel like the brand, not a dead end
```

**Key Principles:**
- Get them back into the funnel quickly
- Don't blame the user
- Use the opportunity to surface your best content
- Make it human and on-brand

## TEMPLATE 10: THANK YOU / CONFIRMATION PAGE

**Best For:** Post-purchase, post-registration, post-opt-in
**Conversion Goal:** Next step action + increase commitment

### Section Structure

**SECTION 1: CONFIRMATION**
```
Headline: "You're in! [Or: Your order is confirmed!]"
Subheadline: "Here's what happens next:"
Details: What they signed up for + confirmation details
Email note: "Check your inbox for [confirmation email]"
```

**SECTION 2: NEXT STEPS**
```
Headline: "While you wait, here's what to do:"
Content: 3 numbered steps
Step 1: Immediate action (check email, join group, etc.)
Step 2: Preparation action (watch intro video, fill profile, etc.)
Step 3: Primary success action (first task toward goal)
```

**SECTION 3: UPSELL OR NEXT OFFER**
```
Headline: "One more thing..."
Content: Relevant next offer
This should feel like a natural next step, not a hard sell
For opt-ins: Offer the next logical resource
For purchases: Offer complementary product
[PLACEHOLDER: Relevant next offer]
```

**SECTION 4: COMMUNITY / CONNECTION**
```
Headline: "Join [X] others [doing the thing]:"
Content: Link to private group, Slack, Discord, or community
This increases retention and reduces refund rates
```

**SECTION 5: SOCIAL SHARING**
```
Content: "Proud of your decision? Share it:"
Share buttons: LinkedIn, Twitter/X, Facebook
Pre-written share copy provided
```

**Key Principles:**
- Celebrate their decision (reduces buyer's remorse)
- Clear next steps reduce anxiety
- Upsell on thank you page converts at 20-30% with right offer
- Community connection increases LTV significantly
